# PRETTIER CODE FORMATTING

Auto-format code with Prettier to ensure consistent code style.

## What to do

1. Run `pnpm format` from project root
2. Prettier will automatically format all files
3. Review changes if significant formatting occurred
4. Report results to user

## Rules

- This command MODIFIES files automatically
- Prettier enforces consistent style (indentation, quotes, etc.)
- No manual intervention needed unless errors occur
- All code should follow Prettier's decisions
- NEVER fight Prettier's formatting

## Expected outcome

All code is formatted consistently according to project Prettier configuration.
